#Portfolio Website
Overview

This project is a personal portfolio website developed to showcase my skills, professional experience, and projects. It serves as a digital resume and provides a structured way for recruiters and employers to evaluate my work.

#About Me

I am Jyoti Navik, a Full Stack Developer with 2 years of experience in building web applications. I previously worked as a System Engineer at Infosys and currently work as a Freelance Web Developer.


#Technologies Used
HTML5
CSS3
JavaScript
Git and GitHub

#Features
Responsive and user-friendly design
Clean and structured layout
Dedicated sections for education, experience, and projects
Project showcase with descriptions
Contact section for communication

#How It Works

The website is built using HTML for structure, CSS for styling, and JavaScript for basic interactivity. Each section of the portfolio is organized using reusable components such as cards to maintain consistency and readability.

The layout is designed using modern CSS techniques like Flexbox to ensure proper alignment and responsiveness across devices.
